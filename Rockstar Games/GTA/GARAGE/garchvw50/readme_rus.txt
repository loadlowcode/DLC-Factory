Game Archive Viewer v5.0
_____________________________

Game Archive Viewer - это утилита для просмотра архивов, используемых различными играми. На данный момент поддерживаются четыре формата:
* IMG для GTA: Vice City
* IMG для GTA: San Andreas
* SYB для Syberia 2
* BIG для Harry Potter and the Goblet of Fire
* PK4 для Prey
* BIN для "Братвы и Кольца"
* PAK для "Санитаров Подземелий"
(архивы пяти последних игр доступны только для чтения)

Внимание! Я не гарантирую, что данная программа будет корректно работать во всех случаях (несмотря на то, что я не нашел явных сбоев в её работе), и я не буду отвечать за её ошибки. Поэтому я ПРОШУ присылать мне письма с сообщениями о подобных ошибках, но все упреки по этому поводу я сочту безосновательными - я вас предупредил. Максимальный вред, нанесенный моей программой вашему компьютеру - неработоспособность редактируемой игры, поэтому по возможности делайте резервные копии всех изменяемых файлов.

_____________________________

Я считаю интерфейс программы достаточно интуитивно понятным :), поэтому опишу только некоторые тонкости по работе с программой.


Правка -> Открыть
Данная команда извлечет выбранный файл в промежуточную папку и попытается его открыть - как если бы это было сделано из Explorer'а. На данный момент моя программа не имеет встроенного просмотрщика упакованных файлов.


Поиск -> Найти все
Выделяет все файлы, соответствующие поисковому запросу и показывает первый из них. Этот пункт становится доступен после использования команды "Поиск -> Найти..."


Поиск -> Фильтр...
Данная опция позволяет отобразить в списке файлов только те, которые удовлетворяют заданной маске.


Сервис -> Перепаковать архив
Аналог "Rebuild archive" в аналогичных программах. Удаляет из архива пустоты, оставшиеся после удаления файлов.
Совет. При редактировании файлов IMG для GTA: VC как правило не обязательно делать резервную копию ВСЕГО архива - достаточно сохранить файл с расширением DIR. Но учтите, что в таком случае производить перепаковку архива (Rebuild archive) нельзя! Этот совет относится не только к моей программе.


Сервис -> Панель просмотра
Загрузка содержимого файлов архива для просмотра в некоторых случаях может вызывать задержки. Если вам не нужно просматривать содержимое файлов, отключите эту опцию.


Сервис -> Сглаживание
По умолчанию при отображении моделей используется сглаживание (антиалиайсинг). Вы можете отключить эту опцию для ускорения работы.


Сервис -> Открывать в GArchVw
При первом запуске программа предложить ассоциировать себя с теми типами файлов, которые она поддерживает. В данном меню также можно выбрать, какие файлы будут открываться при помощи Game Archive Viewer.


Сортировка.
Очень удобным средством может оказаться сортировка списка файлов. Для этого щелкните по заголовку любого из столбцов и перечень будет соответствующим образом пересортирован.


Выделение нескольких файлов.
Game Archive Viewer позволяет выделить несколько файлов сразу - с помощью мыши или клавиш CTRL и SHIFT.


Предварительный просмотр.
Правая панель предназначена для предварительного просмотра файлов некоторых типов, которые могут находится в псевдоархивах. На данный момент поддерживаются следующие форматы:

TXD* (текстуры) для GTA: Vice City и GTA: San Andreas
DFF (модели) для GTA: Vice City и GTA: San Andreas
IPL** (двоичные карты) для GTA: Vice City и GTA: San Andreas
COL (физические параметры моделей, в т.ч. для обработки столкновений) для GTA: Vice City и GTA: San Andreas (поддерживаются все три формата: COLL, COL2 и COL3).
DDS/DXT* (текстуры) для "Братвы и Кольца", "Санитаров Подземелий" и других
TGA* (текстуры) для "Санитаров Подземелий" и других
* изображение из файла текстуры можно сохранить, щелкнув по нему правой кнопкой мыши
** текстовое представление двоичного IPL-файла можно сохранить, щелкнув по нему правой кнопкой мыши

При просмотре файлов DFF, COL, DDS/DXT и DXT камеру можно поворачивать, перемещать и приближать/удалять:
Левая кнопка мыши: вращение (для текстур - перемещение)
Правая кнопка мыши: масштабирование (для текстур - контекстное меню)
Shift+Левая кнопка мыши: перемещение (для текстур - масштабирование)


Новое меню и панель инструментов.
Начиная с версии 5.0 в Game Archive Viewer используется новый стиль меню, а также появилась панель инструментов. Технология нестандартная, поэтому возможны ошибки. В любом случае, панель инструментов можно отключить (Сервис -> Панель инструментов), а меню заменить на старое (Сервис -> Главное меню -> Стандартное Windows). Кроме того, и меню (только новую версию), и панель инструментов можно свободно настраивать, подобно тому, как это делается в MS Office (Сервис -> Настройка...). К сожалению, встроенный диалог на английском языке и перевести его проблематично. Если при настройке вы что-то сделали неправильно или меню по каким-то другим причинам стало работать неверно, можно попробовать вернуть всё "на круги своя", удалив файл actdata.dat - в нем хранятся соответствующие настройки.
